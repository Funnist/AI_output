"RAG_" 로 시작하는 챗봇들은 "DBstore.py" 이 어떤 데이터를 가지고 임베딩을 했느냐에 따라 목적이 바뀌는 파생 챗봇입니다.
내부의 DB 경로를 변경하고 프롬프트를 수정하면 자신의 의도에 맞게 사용하실 수 있습니다.
로컬 LLM을 기반으로 작성이 되었습니다.
위의 챗봇을 도커 이미지로 작성할 수 있는 예제 코드도 포함합니다. 라이브러리와 ollama 로컬 LLM을 포함합니다.
"translate_score.py"는 다국어 번역 평가 코드입니다. 번역업체에서 제공한 엑셀을 입력데이터로 넣으면, 번역에 대한 평가를 진행해 줍니다.
